# O Jardineiro do Bairro - Landing Page

Projeto FrontEnd UC00604 - ATEC TPSI-CAS-0725 

---

## Sobre o Projeto

Landing page desenvolvida para uma empresa de jardinagem e paisagismo. O objetivo foi aplicar técnicas de Front-End e UX/UI num cenário de serviços do mundo real, criando uma experiência web moderna e profissional para uma empresa de arquitetura paisagista.

---

## Tecnologias

- HTML5 (semântica para SEO)
- CSS3 (Flexbox, CSS Grid, @keyframes, custom properties)
- JavaScript Vanilla (lógica de interação)
- Google Fonts (DM Sans, Playfair Display)
- Font Awesome 6.4 (ícones vetoriais)

---

## Estrutura do Projeto

```
├── index.html         - página principal com todas as secções
├── style.css          - estilos personalizados (paleta, animações, layout)
├── script.js          - lógica JavaScript (formulários, smooth scroll, interações)
├── Imagens/           - recursos visuais do projeto
└── README.md          - documentação do projeto
```

# 1. Clonar o repositório ou transferir os ficheiros

---

## Funcionalidades

- Design responsivo (Mobile-first, breakpoints em 768px e 1024px)
- Smooth Scroll para navegação entre secções
- Formulário de contacto com validação de campos obrigatórios
- Botão flutuante do WhatsApp (CTA permanente)
- Animações CSS (pulse, hover effects, transições)
- Secção FAQ com acordeão nativo HTML `<details>`
- Secção de testemunhos com layout em grelha
- Portfolio com overlays informativos
- Newsletter com download de lead magnet
- Menu mobile toggle com JavaScript Vanilla
- Tipografia diferenciada (serifada para títulos, sans-serif para corpo)
- Hero section com imagem de fundo e call-to-action
